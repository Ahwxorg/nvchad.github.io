## Config Structure

```lua
├── init.lua
│
├── lua
│   │
│   ├── core
│   │   ├── default_config.lua
│   │   ├── mappings.lua
│   │   ├── utils.lua  -- (core util functions) (i)
│   │   └── init.lua  -- (autocmds)
│   │
│   ├── plugins
│   │    ├── init.lua -- (default plugin list)
│   │    └── configs
│   │        ├── cmp.lua
│   │        ├── others.lua -- (list of small configs of plugins)
│   │        └── many more plugin configs
|   |
│   ├── custom *
│   │   ├── chadrc.lua -- (overrides default_config)
│   │   ├── init.lua -- (runs after main init.lua file)
│   │   ├── more files, dirs
```
<br/>

-  Note : The custom dir is created by the user.

## Default settings

- This file is lua/core/default_config.lua.
- Chadrc.lua which is in lua/custom/ overrides the default_config file.

```lua
-- Table fields in default_config.lua
M.ui = {},
M.plugins = "",
M.mappings = {},
M.lazy_nvim = {}, -- startup config for lazy.nvim
```
