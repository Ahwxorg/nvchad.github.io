title: Post Install
---

- Some things must be done after the installation.

### Install a LSP server

- Example : 
```
:LspInstall css 
```

### Install a treesitter parser

- Example : 
```
:TSInstall css
```

### Set a theme

- By default NvChad comes with many themes.
- Switch themes with this mapping : 

```
<leader> + th
```
